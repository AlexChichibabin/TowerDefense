# TowerDefense

## Реализовано:
### На уровне:
1. Возможность ставить башни на специальные слоты и удалять поставленные башни
2. Разнообразие типов башен (8) с разными свойствами атаки
3. Маршруты для передвижения противников
4. Разнообразие типов врагов (8) с разными скоростью передвижения и брони
5. Возможность гибко настраивать волны по типу и количеству врагов
6. Урон игроку за каждого врага на нашей базе
7. Игрок может преждевременно вызвать следующую волну получив золото, пропорциональное остатку времени до следующей волны
8. Отображение жизней и маны
9. Можно использовать заклинания (2), чтобы нанести урон врагам или замедлить их
10. Анимации и звуки
11. Меню паузы
12. Возможность выиграть или програть

### Прогрессия и меню:
1. Есть система прогрессии и сохранения
2. Разветвленная система уровней. Есть главная линия уровней и бонусные уровни на боковых ветках
3. В зависимости от успешности прохождения уровня игрок получает 0-3 звезды
4. В меню есть магазин башен. В них можно тратить звезды на улучшение башен
5. Анимации и музыка в меню
6. Возможность сбросить прогресс начав новую игру
7. Несколько играбельных уровней(5)
